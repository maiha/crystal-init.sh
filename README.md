# crystal-init.sh

a simple wrapper for crystal init command 

- automate my initial jobs

## Usage

```shell
crystal-init.sh lib XXX
```

## Features

- adjust names between `XXX` and `XXX.cr`

#### README.md

- add `description`
- add `crystal version`
- add `github username`

#### .gitignore

- add `/.crystal-version`

#### src

- delete `src/XXX/version.cr`
